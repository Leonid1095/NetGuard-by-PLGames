# ✨ DNS Shield Pro - Новый простой интерфейс

## 🎯 Главное изменение

Теперь DNS Shield Pro работает как **WireGuard Portal** - **"нажал кнопку и работает"**!

## 📱 Как использовать

### Первое подключение:

1. **Выберите сервер** (если нужно):
   - Нажмите на кнопку с названием сервера
   - Откроется список всех доступных серверов
   - Выберите нужный сервер

2. **Нажмите большую кнопку ПОДКЛЮЧИТЬ**
   - Всё! Вы подключены
   - Круг статуса станет синим (FREE) или золотым (VIP)
   - Кнопка изменится на "ОТКЛЮЧИТЬ"

3. **Готово!**
   - Статистика показывает запросы и блокировки
   - DNS Shield защищает ваше соединение

### Использование Public DNS:

1. На главном экране нажмите **"🌐 Public DNS"**
2. Выберите DNS провайдера
3. Следуйте инструкциям для вашей платформы

### Использование Tools:

1. На главном экране нажмите **"🔧 Tools"**
2. Выберите нужный инструмент
3. Используйте его функционал

## 🎨 Что нового?

### ✅ Добавлено:
- **Большой круг статуса** - сразу видно подключены вы или нет
- **Огромная кнопка подключения** - её невозможно не заметить
- **Анимация пульсации** - когда подключены
- **Быстрый выбор сервера** - всего 2 клика для смены
- **Overlay экраны** - Public DNS и Tools не загромождают главный экран
- **Плавные анимации** - всё работает красиво и быстро

### ❌ Убрано:
- Три вкладки (Private DNS, Public DNS, Tools)
- Сложная навигация
- Маленький статус

### 💡 Осталось:
- Весь функционал Public DNS (8 зашифрованных провайдеров)
- Все инструменты Tools (8 штук)
- Настройки, помощь, информация
- FREE и VIP версии
